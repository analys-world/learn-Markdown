# 一.Markdown 是什么？

Markdown是一种轻量级的标记语言，它是以纯文本形式来编写文档，并最终以HTML的形式发布。

# 二.为什么要使用Markdown

markdown易读，易写，易修改，十分的方便。
可以转换为HTML格式发布。
跨平台使用。

# 三.使用方法

Markdown的语法主要包含以下及大部分：

**标题** **段落** **区块** **代码** **强调** **列表** **分割线** **链接** **图片** **反斜杠** **符号**。

**标题**

使用#，可以表示不同等级的标题

    # 一级标题
    ## 二级标题
    ### 三级标题
    #### 四级标题
    ##### 五级标题

**段落**

段落的前后要有空行，所谓的空行就是指的是没有文字内容，要是想在段内强制换行的方式就是使用两个以上的空格加上回车(引用中换行省略回车)

**区块**

在段落的每行或者只在第一行使用符号> , 还可以使用多个嵌套引用，比如：

    > 区块
    >> 嵌套

> 区块
>> 嵌套




**代码**

代码的建立是在每行加上四个空格或者一个制表符(如同写代码一样)，如普通段落：


    void main()
    {
        print("hello world");
    }

**强调**

只要在强调的内容两侧分别加上*或者_ , 

*斜体* ， _斜体_
**粗体** ， __粗体__

**列表**

使用. + - 标记无序列表， 如

- 第一项
- 第二项
- 第三项


**分割线**

分割线最常使用的就是三个或以上的*， 还可以使用-和_.

**链接**

链接可以由两种形式生成：行内式与参考式

行内式：
[1](1)


**图片的形式**

图片的形式与链接相似，只需要在链接的基础上前方加一个！

**反斜杠\ **

相当于反转义作用，使符号成为普通符号

**符号**

起到对应的标记作用，突出行文重点
